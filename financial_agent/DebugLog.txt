### 📊 第二步：建立樞紐分析表（以 Excel 為例）
1. 將匯出的檔案開啟於 Excel，確保 **第一列為欄位標題**。
2. 選取整份資料 → 上方選單點擊 `插入` → `樞紐分析表` → 選擇「新工作表」→ 確定。
3. 在右側「樞紐分析表欄位」面板中，依需求拖曳：
   - **列**：`ProductCategory`（或您想分析的維度）
   - **值**：`TotalAmount`（預設為「求和」，若顯示為「計數」請右鍵→值欄位設定→求和）
   - **篩選器**：`OrderDate`（可預設勾選 2014 年）
   - **行**（選填）：`Region` 或 `Month`（若需交叉分析）
4. 格式化建議：右鍵數值 → `數值格式` → 設定為貨幣；可加入「排序」與「欄位設定」提升可讀性。

### 🐍 補充：Python（pandas）自動化建立樞紐表
若您習慣使用程式處理，以下為簡易範例：
```python
import pandas as pd
df = pd.read_csv('2014_sales_draft.csv')

# 建議先清洗：移除異常值、轉換日期格式
df['OrderDate'] = pd.to_datetime(df['OrderDate'])
df_2014 = df[df['OrderDate'].dt.year == 2014]

pivot = pd.pivot_table(
    df_2014,
    values='TotalAmount',
    index='ProductCategory',
    columns='Region',
    aggfunc='sum',
    fill_value=0
)
pivot.to_excel('2014_Sales_Pivot.xlsx')
```
